# TASK MANAGER API

A simple task manager api which gives functionality to create, list, update and delete tasks.

## Prerequisites

- Node.js 18+

## Installations

**Clone the repository**

```bash
git clone https://github.com/airtribe-projects/task-manager-api-rohitbaisane.git

Install Dependencies

npm install
```

**Run the project**

```bash
#Run tests

npm test

#start the server

npm start
```
